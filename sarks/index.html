What is DNS?

DNS stands for Domain Name System. It is a system used to translate domain names, which are easy-to-remember words or phrases used to identify websites and other online resources, into IP addresses, which are the numerical addresses used by computers to locate these resources on the internet. When you type a domain name into your web browser, your computer sends a request to a DNS server to look up the IP address associated with that domain name. The DNS server then responds with the correct IP address, allowing your computer to connect to the website or resource you requested.

  • Security: Your Internet service provider(ISP) is most likely your DNS provider meaning that they can see what you do and every site you visit.
  • Privacy: Your activity traces can be used to target you with ads.
